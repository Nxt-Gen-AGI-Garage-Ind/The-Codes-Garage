## 🏗️ System Overview
**The Code Garage** is a serverless, agentic workspace designed for high-performance code diagnostics. Built for freelancers and developers who need a "No-Cap" environment to fix, tune, and debug code using a variety of open-source and free-tier AI models.

## 🛠️ Tech Stack & Integrations
* **Frontend:** React / Vite / Tailwind CSS (Industrial Theme)
* **Intelligence:** * **The Foreman:** Smart AI Selector logic.
    * **Universal API:** OpenAI-compatible base URL integration.
    * **Local Power:** Dedicated **Ollama** & **Puter.js** modals.
* **Hosting:** Fully static deployment for **GitHub Pages**.

## 🚀 Ignition (Getting Started)
1.  **Clone the Bay:** `git clone https://github.com/Nxt-Gen-AGI-Garage-Ind/The-Codes-Garage.git`
2.  **Install Tools:** `npm install`
3.  **Start the Engine:** `npm run dev`
4.  **Configure:** Open the **Gear Icon** in the UI to add your API keys (Stored locally).
